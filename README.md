# MouseAnchor
WoW Addon to Anchor tooltip to the mouse

# How to install
- Download newest release from [releases](https://github.com/Empyreum/MouseAnchor/releases)
- Unzip content to your AddOns directory. Its path is (ex.: C:\World of Warcraft\_retail_\Interface\AddOns).
- Done! Restart WoW or reload ui and your new addon should show in the AddOns list on your character select screen.

## Get current version number (to update addon)
```
/dump select(4, GetBuildInfo())
```
